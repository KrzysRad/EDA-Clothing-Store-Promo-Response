# Customer Response to Promotions - Data Mining Project

## Project Overview

This project focuses on the analysis of customer purchasing behavior in a clothing retail store and the prediction of customer responses to promotional campaigns.

The main objective of the analysis was to identify the factors influencing customer engagement with promotions and to compare the performance of several machine learning classification models.

The project includes:
- exploratory data analysis (EDA),
- data cleaning and preprocessing,
- visualization of customer behavior patterns,
- correlation analysis,
- feature engineering,
- predictive modeling and model comparison.

---

## Dataset

The dataset contains information about:
- customer purchasing activity,
- spending behavior,
- product preferences,
- promotional history,
- customer engagement indicators.

The data originates from educational materials accompanying a publicly available book on predictive modeling and data mining.
